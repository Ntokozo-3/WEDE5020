# WEDE5020

This project is about a website that consists of five webpages, they are as follows: Home page which is the landing page, about page which tell people more about my business or company, product page which consist of the things we offer, inquiry page having a form to fill in order to be in contact with us if you need anything, and finally the contact page which has the company's information, email and phone numbers.

This project is designed using the HTML AND CSS, implementing the CSS makes the website looks appealing to the eyes of people and the structure of the website makes it easier for people to navigate through it.
